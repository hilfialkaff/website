Accelerometer Biometric Competition

We enroll in Kaggle's Accelerometer Biometric Competition in which we are being given 60 million unique samples of accelerometer data collected from 387 different devices. The goal of this competition is to recognize users of mobile devices from accelerometer data.

Chat Formality

In this project, we analyze the formality of instant messaging chats as we know a person. Formality in this project is being naively determined by the number of occurance of dictionary based words in the chat.

CrowdWatch

CrowdWatch is a novel crowdsourcing mechanism that take advantage of two distinct radio that exist in phones nowadays, i.e. Bluetooth, which is short-ranged but more energy-conserving, and WiFi, which is a longer range radio but is more energy-draining.

Publication

  • CrowdWatch: enabling in-network crowd-sourcing.
    Robin Kravets, Hilfi Alkaff, Andrew Campbell, Karrie Karahalios, Klara Nahrstedt
    SIGCOMM MCC 2013

Context-Centric Security

Design a system for Android that spans smart clients and datacenter servers, and allows end-users to intuitively control their private data while developers use an API to create secure program.

Tessellation OS

In this project, I implemented resource guarantees policies for the graphic subsystems and support for ACPI and PCI devices discovery in Tessellation. I have also helped in porting multiple applications such as Intel TBB and video players from Linux to Tessellation.

Publication

    CATA 2012 Best paper.
